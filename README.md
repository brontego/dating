# Dating

This repository is a lightweight dating and Hinge support system.

It exists to reduce overwhelm, not to turn dating into another full-time admin job.

## Current operating idea

- keep profile, screening, and messaging support in one place
- make swiping and messaging more intentional
- preserve what you learn about attraction, fit, and dead ends
- keep dating reentry bounded so it does not swallow the rest of life

## Core files

- `state/current_week.yaml`
- `state/subsystem_snapshot.yaml`
- `deliverables/current/hinge_reboot.md`
- `deliverables/current/profile_edits.md`
- `deliverables/current/opener_queue.md`
- `deliverables/current/reply_drafts.md`
- `deliverables/current/screening_summary.md`

## Ground rule

The system should help you date more calmly and honestly.
If it starts creating pressure, it needs to get simpler.
